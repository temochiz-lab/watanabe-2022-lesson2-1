# watanabe-2022-lesson2-1
2022年度授業用プログラム2セッション1

# 概要
指定秒数を表示する。　指定秒数になったと思ったところでキーを押す。

# プログラムの動作 
1.学籍番号、性別、年齢の入力  
2.実験開始のメッセージ  
3.凝視点 + 音声(ピッピッピッピー) 
4.指定秒数を文字で表示 例)1.5秒  
5.指定秒数になったと思ったところで何かキーを押す  
6.(3-5を繰り返し)  
7.終了メッセージ  

# インストール方法
jsPsych7 をインストールしたディレクトリ内に、コピーしておきます。  
こんなディレクトリ構成になります。  
  
jsPsych7  
-- dist  
-- examples  
-- このプログラムのディレクトリ

# 実行方法
index.html のクリックで実行します。
